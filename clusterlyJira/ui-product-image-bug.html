UI showing same image for all recommended products

Medium Priority
Description

Bug Summary: In the "Recommended for You" section (Cart and Product pages), all product items are displaying the same image (/assets/images/deals/mobile.webp) regardless of the actual product type.

Expected Behavior: Each recommended product should display its unique product image corresponding to the specific item (headphones, fitness watch, power bank, charger).

Actual Behavior: All 4 recommended products show the same mobile phone image.

‚ö†Ô∏è User Experience Impact
  • Poor visual representation of product diversity
  • Confusing user experience - users may think all products are the same
  • Reduced trust in product recommendations
  • Potential decrease in recommendation click-through rates
Root Cause Analysis

The issue is in the RecommendedProducts.tsx component where all mock products are hardcoded to use the same image path:

store-ui/src/components/Cart/RecommendedProducts.tsx (lines 20-50)

Proposed Solution

Update the image paths in the mock data to use appropriate images for each product type:

Suggested fix for RecommendedProducts.tsx
const defaultProducts: RecommendedProduct[] = [
    {
        id: '1',
        title: 'Wireless Bluetooth Headphones',
        price: 2999,
        image: '/assets/images/deals/headphones.webp',  // ‚úÖ Appropriate image
        vendor: 'TechBrand'
    },
    {
        id: '2', 
        title: 'Smart Fitness Watch',
        price: 5999,
        image: '/assets/images/deals/watch.webp',       // ‚úÖ Appropriate image
        vendor: 'FitTech'
    },
    {
        id: '3',
        title: 'Portable Power Bank', 
        price: 1499,
        image: '/assets/images/deals/powerbank.webp',   // ‚úÖ Appropriate image
        vendor: 'PowerPlus'
    },
    {
        id: '4',
        title: 'USB-C Fast Charger',
        price: 899,
        image: '/assets/images/deals/charger.webp',     // ‚úÖ Appropriate image
        vendor: 'ChargeMax'
    }
];

Alternative Solution: If specific product images are not available, use existing diverse images from the deals folder:

  • Headphones: /assets/images/deals/oven.webp
  • Fitness Watch: /assets/images/deals/kurtha.webp
  • Power Bank: /assets/images/deals/shoes.jpg
  • Charger: Keep /assets/images/deals/mobile.webp
